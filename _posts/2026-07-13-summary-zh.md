## [Claude Code 与 OpenCode：令牌效率研究显示 33k 对比 7k 令牌](https://systima.ai/blog/claude-code-vs-opencode-token-overhead) ⭐️ 8.0/10

一项比较 AI 编码工具 Claude Code 和 OpenCode 的技术研究发现，Claude Code 消耗 33,000 个令牌，而 OpenCode 仅使用 7,000 个令牌来完成类似的编码任务。该研究通过在代理编码工具和 Anthropic 端点之间添加日志记录来捕获所有请求和用法数据。 这一发现对关注 AI 工具成本的开发者和组织具有重要意义，因为它揭示了流行编码助手之间令牌效率的显著差异。4.7 倍的令牌使用差异可能转化为广泛使用这些工具的团队的重大成本影响。 该研究特别指出，Claude Code 的低效源于其缓存策略和工具令牌使用。研究人员注意到其方法学的一个注意事项，并计划更新文章，包含更深入的任务、定性结果以及输入和输出的复现。

hackernews · systima · 7月12日 18:25 · [社区讨论](https://news.ycombinator.com/item?id=48883275)

**背景**: 像 Claude Code 和 OpenCode 这样的 AI 编码工具使用大型语言模型（LLM）来帮助开发者完成编码任务。这些工具消耗令牌，即 LLM 处理的基本文本单位，每个令牌使用都会产生成本。随着组织采用这些 AI 助手，令牌效率已成为一个关键考虑因素，不同的实现方式在管理编码任务期间的令牌消耗方面显示出不同程度的优化。

## [新型钠金属电池设计可在 4 分钟内充电并保持多年容量](https://www.reddit.com/r/technology/comments/1uufxgh/new_sodium_metal_battery_design_charges_in_just_4/) ⭐️ 8.0/10

中国研究人员开发出一种新型钠金属电池设计，可在 4 分钟内从零充至满电，同时保持多年容量，相关发现已发表在《Nano-Micro Letters》期刊上。 这一突破可能彻底改变储能技术，通过提供比当前锂离子电池更快充电、更耐用的替代方案，可能影响电动汽车和可再生能源系统。 新电池设计解决了钠金属电池的关键挑战，包括枝晶生长和不稳定的固体电解质界面（SEI）层，实现了超快充电能力和 6000 小时稳定性，如研究人员所报告。

reddit · r/technology · /u/lurker_bee · 7月12日 13:45

**背景**: 钠离子电池（NIBs）是使用钠离子作为电荷载体的可充电电池，类似于锂离子电池，但用钠替代锂。它们提供了潜在的成本优势，并作为锂离子技术的替代方案被探索，特别是用于大规模储能应用。钠金属电池（SMBs）特别旨在实现更高的能量密度，但面临枝晶形成和稳定性等挑战，而这项新设计似乎解决了这些问题。
