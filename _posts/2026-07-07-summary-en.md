## [Ternlight – 7 MB embedding model that runs in browser (WASM)](https://ternlight-demo.vercel.app/) ⭐️ 8.0/10

A 7MB sentence embedding model called Ternlight has been developed that runs entirely in the browser using WebAssembly, enabling on-device text similarity calculations without requiring server dependencies. This achievement is significant because it demonstrates the feasibility of running sophisticated AI models directly in web browsers, which could enable new privacy-preserving applications, reduce server costs, and improve user experience by eliminating network latency for text similarity tasks. The model was created by distilling a small sentence encoder from MiniLM using ternary quantization-aware training, with the inference engine written from scratch in Rust and compiled to WebAssembly with SIMD optimizations. It produces 384-dimensional vectors where cosine similarity between vectors determines text relatedness.

hackernews · soycaporal · Jul 6, 23:06 · [Discussion](https://news.ycombinator.com/item?id=48811644)

**Background**: Embedding models are machine learning models that transform diverse inputs like text into fixed-length dense vector representations called embeddings. These embeddings position semantically similar items closer together in a high-dimensional space. WebAssembly (Wasm) is a portable binary code format designed for high-performance applications on web pages, allowing code compiled from languages like Rust to run efficiently in browsers.

## [Tencent Announces Hy3: 295B-Parameter Open-Source MoE Model](https://simonwillison.net/2026/Jul/6/hy3/#atom-everything) ⭐️ 8.0/10

Tencent has announced Hy3, a 295B-parameter Mixture-of-Experts (MoE) model with 21B active parameters and 3.8B MTP layer parameters. The model outperforms similar-size models and rivals flagship open-source models with 2-5x parameters, following feedback from 50+ products and scaled-up post-training with higher quality data. This is significant because it represents a major technical achievement in AI/ML, particularly in the development of efficient large language models. The Apache 2.0 license and free availability make it highly relevant to the AI research community, potentially accelerating innovation and enabling broader access to state-of-the-art AI capabilities. The full-sized Hy3 model is 598GB on Hugging Face, with an FP8 quantized version available at 300GB. It has a context length of 256K and is available for free on OpenRouter until July 21st. The model shows significant gains in utility across various products and productivity tasks.

rss · Simon Willison · Jul 6, 23:57

**Background**: Mixture-of-Experts (MoE) models are a type of AI architecture that divides a large model into multiple specialized sub-networks, where only a small subset activates per token. This allows for the knowledge of a massive model at the inference cost of a much smaller one. Notable MoE models include Mixtral 8x7B (Mistral AI), Google's Gemini 1.5, and reportedly GPT-4. The technique is increasingly common in state-of-the-art large language models, enabling efficient scaling beyond traditional transformer architectures.
